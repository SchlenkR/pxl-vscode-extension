# Changelog

## [0.1.0] - 2026-02-26

Initial release.

- Built-in simulator host with auto-start (bundled for macOS, Linux, Windows)
- Live LED matrix preview in sidebar and editor panel
- Flat and Clock rendering modes
- Run, stop and restart pixogram scripts from the editor toolbar and file tree
- File explorer for clock faces, demos and custom pixograms
- Hot reload on file changes
- Output log with simulator host console output
- Automatic retry when simulator is still starting up
